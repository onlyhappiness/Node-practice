## Features

- 런타임
- 이벤트 기반
- 논 블로킹 I/O
- 싱글 스레드 모델

```
Node.jsⓇ는 Chrome V8 Javascript 엔진으로 빌드된 Javascript 런타임입니다.
```

여태까지 나는 Node를 서버만 실행하기 위해 사용했지만,
Node는 서버 외의 자바스크립트 프로그램을 실행하는 런타임으로서 사용할 수 도 있다.

<br>
